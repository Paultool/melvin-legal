Política de Privacidad

Esta aplicación, REVOLVER TT, utiliza la API de TikTok para permitir al usuario subir y agendar sus propios videos.

Datos que recopilamos

No recopilamos ni almacenamos datos personales en servidores externos. La aplicación funciona de forma local en el ordenador del usuario. Los tokens de acceso de TikTok se almacenan únicamente en el archivo local .env del usuario.

Uso de la información

Los permisos solicitados (video.upload, video.publish) se utilizan exclusivamente para realizar la función de publicación solicitada por el usuario.

Contacto

Para dudas, contactar a través del dominio del desarrollador.
